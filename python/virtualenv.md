# Virtualenv

## Install virtualenv
```bash
pip3 install virtualenv
```

### Create new environment
```bash
cd ~/Environment

mkdir py27_env
virtualenv py27_env
virtualenv -p /usr/bin/python2.7 py27_env/
```

#### Activate python environment
```
bash~/Environment/py27_env/bin/activate
```

#### Deactivate python environment
```bash
deactivate
```
